# Projeto C.I.C. (Nome Temporário)

## Descrição
Sistema de proteção financeira inteligente desenvolvido para motoristas de aplicativo. O projeto foca em "blindar" o lucro do dia a dia, utilizando inteligência artificial para auxiliar o usuário a atingir suas metas financeiras com disciplina.
