# Projeto C.I.C. (Nome Temporário)
​O C.I.C. é uma plataforma financeira desenvolvida especificamente para a realidade de motoristas de aplicativo. Utilizando princípios de economia comportamental, o projeto visa ajudar o motorista a atingir metas de longo prazo (como a troca de veículo) através de fricção cognitiva, suporte emocional e uma interface adaptada para o trânsito.

​🚀 O Problema
​Motoristas de app enfrentam a dificuldade de poupar devido à natureza do fluxo de caixa diário e à facilidade de saque imediato. O C.I.C. inverte essa lógica, transformando a transação financeira em um processo consciente.
​🛠 Funcionalidades Principais
​Saque Protegido (Protocolo 5 Etapas): Jornada de fricção cognitiva que força o usuário a confrontar o custo de oportunidade (atraso na meta e impacto emocional) antes de sacar valores.  
​Coach Comportamental (IA): Treinador financeiro baseado em LLM com suporte para o modo mãos livres, ideal para uso durante a jornada de trabalho.  
​Gestão de Metas: Acompanhamento de progresso focado no esforço real do motorista (horas de corrida necessárias).  
​💻 Tecnologias Utilizadas
​Frontend: React.js
​Inteligência Artificial: Integração via API para suporte comportamental e estratégico.  
​Voz: Web Speech API para interação segura.
​📋 Natureza da Parceria
​Este projeto é regido por um Contrato de Parceria Estratégica com Participação nos Resultados, estabelecendo uma colaboração mútua para o desenvolvimento, crescimento e monetização dos negócios digitais gerados pela plataforma. A parceria envolve:  
​Direcionamento e Estratégia: Planejamento de crescimento e mentoria.  
​Execução Operacional: Implementação técnica e administrativa do projeto.  
​Participação Econômica: Divisão igualitária (50/50) dos resultados obtidos.  
​💡 Propriedade e Confidencialidade
​Todas as estratégias, métodos e sistemas desenvolvidos nesta parceria são protegidos por direitos de propriedade intelectual, sendo mantido o dever de sigilo permanente entre os envolvidos. O projeto tem vigência por prazo indeterminado enquanto perdurarem os resultados econômicos gerados.  
​Desenvolvido com foco em escalabilidade digital e sucesso financeiro do usuário final.
